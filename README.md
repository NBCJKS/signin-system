# 签到积分系统

一个功能完整的签到积分管理系统，支持用户签到、积分商城、管理后台等功能。

## 在线访问

部署后访问：`https://你的用户名.github.io/签到积分系统/`

## 功能特性

### 👤 用户端
- ✅ 用户登录（仅需输入名称）
- ✅ 每日签到获得积分
- ✅ 积分商城兑换
- ✅ 签到记录查看
- ✅ 积分排行榜

### ⚙️ 管理后台
- ✅ 签到管理（设置规则、修改用户签到天数）
- ✅ 惩罚扣分 / 获奖加分
- ✅ 用户信息查询
- ✅ 积分商城管理
- ✅ 白名单管理
- ✅ 兑换商品管理
- ✅ 系统维护
- ✅ 代为签到
- ✅ 多级管理员

### 📱 PWA 功能
- ✅ 可安装到手机桌面
- ✅ 离线使用
- ✅ 全屏体验

## 部署说明

### 方法一：GitHub Pages（推荐）

1. 创建新仓库，仓库名建议：`signin-system`
2. 上传所有文件
3. 进入 Settings → Pages
4. Source 选择 `main` 分支
5. 等待部署完成即可访问

### 方法二：Vercel

1. 访问 vercel.com
2. 用 GitHub 登录
3. Import 项目
4. Deploy 完成

### 方法三：Netlify

1. 访问 netlify.com
2. Drag & Drop 上传文件夹
3. 完成部署

## 技术栈

- 纯前端实现（HTML + CSS + JavaScript）
- LocalStorage 数据持久化
- PWA 离线支持
- 响应式设计
